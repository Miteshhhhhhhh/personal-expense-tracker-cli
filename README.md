# Personal Expense Tracker CLI

Python menu-based expense tracker with persistent Excel storage.

## Features
- Add new expenses with category + amount + date
- View all saved expenses
- Calculate total spending
- Filter expenses by category
- Export filtered results to Excel
- Data persists between runs

## Tech Stack
- Python 3
- Pandas
- Openpyxl

## How to Run
1. `pip install -r requirements.txt`
2. `python expense_tracker.py`

## Key Learning
Built CRUD operations and file-based persistence using pandas DataFrame as database.
